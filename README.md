# Contacts table app

> Приложение, которое использует randomuser API для отображения полученных данных в таблице.

## Ссылка для просмотра

[Live Demo Link](https://contacts-table-app-eea30.web.app/) :point_left:

Присутствующий функционал:
- Обновление пользователей в таблице по нажатию кнопки.
- Поиск по полному имени с небольшой валидацией.
- Фильрация пользователей по гендеру и национальности (можно использовать как по отдельности, так и вместе)
- Очистка фильтров по нажатию кнопки.
- Сортировка пользователей по имени (2 варианта сортировки: от а до z и от z до a)

### Использование
1. Склонировать / скачать репезиторий.
2. Подгрузить пакеты (node_modules) одной из двух команд (`npm i` или `yarn`)
3. Запустить проект (`yarn start` или `npm run start`)
